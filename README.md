# Christmas Tree Kata

```
drawTree(5):

    X
   XXX
  XXXXX
 XXXXXXX
XXXXXXXXX
    |
```

The `size` defines the rows of leaves, excluding the tree stump.

If `size` is less than or equal to zero, then draw only the tree stump.
